# Design Skills Hub — Logo

## Files

| File | Usage |
|------|-------|
| `logo-dark.svg` | Dark backgrounds, main brand mark. Vapor Purple (#9D84BA) nebula + glow. |
| `logo-light.svg` | Light backgrounds, transparent fill. Line art version in #9D84BA. |

## Colors
- Primary: `#9D84BA` (Vapor Purple)
- Light accent: `#C4AED6`
- Dark accent: `#7A6391`

## Usage Notes
- `logo-dark.svg` — use on dark/colored backgrounds
- `logo-light.svg` — use on white/light backgrounds
- Both are SVG (scalable, no quality loss at any size)
